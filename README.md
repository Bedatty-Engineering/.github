# Bedatty Engineering `.github`

Organization-wide default files for GitHub.

This repository centralizes:

- the organization's public profile;
- default issue and pull request templates;
- community files shared across repositories.

## Structure

- `profile/README.md`: organization's public README.
- `.github/ISSUE_TEMPLATE/`: default issue forms.
- `PULL_REQUEST_TEMPLATE.md`: default pull request template.
- `CONTRIBUTING.md`: short contribution guide.
- `SUPPORT.md`: expected support channels.
- `CODEOWNERS`: default ownership.
