Public workspace for **personal infrastructure and software**—homelab energy, side projects, and learning in the open. Nothing here is a product roadmap; it is a place to ship ideas, break them gently, and leave breadcrumbs for anyone curious.

## What shows up

Repositories span whatever is interesting at the moment: reusable automation, cloud and edge tooling, small apps, and experiments that might graduate into something sturdier—or stay happily rough. Quality varies on purpose; usefulness and clarity matter more than polish.

## If you are browsing

You are welcome to read, fork, or borrow patterns that fit your context. If something helps, a star or a short note is always appreciated. If something breaks, open an issue on the **repository that contains the code or docs** so the thread stays in one place.

## If you want to contribute

Not every repo is looking for drive-by changes; some are personal sandboxes. When contributions make sense, each project points to expectations in its own readme. Org-wide norms (licensing heuristics, how to propose changes) live in [`CONTRIBUTING.md`](../CONTRIBUTING.md). For help, scope, and issue etiquette, see [`SUPPORT.md`](../SUPPORT.md).
